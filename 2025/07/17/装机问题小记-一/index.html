装机问题小记 一

2025/7/17 问题解决 装机

前言

最近，朋友购买了电脑配件，在装机过程中碰到了一些问题:
使用winpe安装Win11 24H2专业版后开机重启，提示：

windows 安装程序无法将 windows 配置为在此计算机的硬件上运行 / Windows无法完成安装。若要在此计算机上安装 Windows，请重新启动安装

然后多次重新安装后无果，从网上搜集到一些解决办法，尝试后成功。


我首先是怀疑是不是因为硬盘分区类型问题，修改后仍无法安装

此机器部分配置如下：

  • CPU：AMD Ryzen 5 5600
  • 内存：16GB DDR4
  • 主板: 华硕A520M-K

解决方法原地址链接：bilibili (这个链接应该是失效了，方法是在评论区找到的)
此方式是由B站热心网友@南国椰子鸡提供，在此非常感谢

修复方法就是在报错界面通过修改注册表，强行跳过初始化安装步骤

修复方法

一.新建用户

  1. 首先新建一个拥有管理权限的账户，在报错界面(如图1)按住键盘上的shift+F10键打开cmd窗口，
    然后输入以下命令(请将用户名替换为自己的想要的用户名)
net user 用户名 /add
 
net user 用户名 /active
 
net localgroup administrators 用户名 /add
  1. 然后在cmd中输入以下命令打开注册表
regedit

因为是因为在系统加载界面报错，所有有可能鼠标指针无法显示，可通过键盘上的TAB键切换目录框，然后通过方向键切换选项，回车确定

  1. 然后定位到HKEY_LOCAL_MACHINE/SYSTEM/setup文件夹下
  2. 将所有数据表全部修改为0，不管什么类型
  3. 修改完后执行以下命令重启应用更改
shutdown -r -t 0

重启后系统成功安装启动

问题解决

  1. 启动后黑屏:

    按住键盘上的Ctrl+Alt+Del键选择登出，然后重新登录账户。(重启应该也行)

  2. 开机后无法关机

    在bios中将快速启动关闭，保存退出重启即可
